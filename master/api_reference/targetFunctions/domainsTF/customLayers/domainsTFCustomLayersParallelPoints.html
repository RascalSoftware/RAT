domainsTFCustomLayersParallelPoints

PURPOSE

Single threaded version of the custom layers, domainsTF reflectivity

SYNOPSIS

function [outSsubs,backgs,qshifts,sfs,nbas,nbss,resols,chis,reflectivity,Simulation,shifted_data,layerSlds,domainSldProfiles,allLayers,allRoughs] = domainsTFCustomLayersParallelPoints(problemDef,problemDefCells,problemDefLimits,controls)

DESCRIPTION 

 Single threaded version of the custom layers, domainsTF reflectivity
 calculation. The function extracts the relevant parameters from the input
 arrays, allocates these on a pre-contrast basis, then calls the 'core' 
 calculation (the core layers standardTf calc is shared between multiple
 calculation types).
